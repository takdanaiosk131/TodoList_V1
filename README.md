<h1>TODOLIST APP</h1>
ToDo list is a simple web application to save Your daily tasks in order not to miss anything.

<h2>Features</h2>
<ul>
    <li>Show task</li>
    <li>Add new task</li>
    <li>Edit task</li>
    <li>Delete task</li>
</ul>

<h2>Technologies</h2>
<ul>
    <li>React</li>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript(ES)</li>
    <li>Bootstrap 4</li>
</ul>
